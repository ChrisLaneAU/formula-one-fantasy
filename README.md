# Formula 1 Fantasy Game

A Ruby on Rails web app that simulates the 2019 Formula 1 Fantasy league.

### Demo

[Demo](https://f1-fantasy-chris-lane.herokuapp.com/)

### Tech Stack

Ruby on Rails + jQuery + GSAP
